# 📸 SnapSpace – Virtual Photobooth

SnapSpace is an interactive web-based virtual photobooth that enables users to capture photos using their device camera, apply real-time visual filters, and generate downloadable photostrips.

## 🚀 Features
- 📷 Live webcam access using WebRTC
- 🎨 Real-time filter effects (grayscale, sepia, invert, etc.)
- ⏱️ Countdown timer for hands-free capture
- 📸 Multiple photo capture
- 🧾 Photostrip generation (combine multiple images)
- 💾 Download images instantly

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- WebRTC (getUserMedia API)
- Canvas API

## 💡 Highlights
- Built a real-time camera-based application directly in the browser
- Implemented image processing using Canvas API
- Designed a clean and aesthetic user interface for better user experience
